# sf-quant-dashboard
Performance analysis dashboard for Silver Fund quant team.

## Prerequisites
- Install [UV](https://docs.astral.sh/uv/getting-started/installation/)

## Usage
Run the dashboard in app mode with sandbox dependencies. Sandbox environments inline the Python package dependencies necessary to run the dashboard.

```python
uvx marimo run dashboard.py
```

## Development

Run the dashboard in edit mode with a sandbox environement. Sandbox environments inline the Python package dependencies necessary to run the dashboard.

```python
uvx marimo edit dashboard.py --sandbox
```
